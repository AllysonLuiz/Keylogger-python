# Teoria: Keyloggers em Segurança da Informação

## O que são Keyloggers?
Keyloggers são programas capazes de registrar as teclas digitadas em um dispositivo.  
Eles podem ser usados de forma **maliciosa** (roubo de credenciais, espionagem) ou **legítima** (monitoramento corporativo autorizado, testes de segurança).

## Tipos de Keyloggers
- **Baseados em software**: Instalados como programas que capturam eventos de teclado.
- **Baseados em hardware**: Dispositivos físicos conectados entre teclado e computador.
- **Baseados em API**: Utilizam funções do sistema operacional para interceptar entradas.
- **Baseados em kernel**: Operam em nível mais profundo, com maior capacidade de ocultação.

## Objetivo Educacional
Este projeto demonstra um keylogger simples em Python para:
- **Entender como funcionam ataques reais**
- **Treinar a detecção e resposta**
- **Desenvolver contramedidas eficazes**

⚠️ **Aviso**: O uso indevido de keyloggers é ilegal. Este material é apenas para estudo.
